## 実験の種類

このプロジェクトでは様々な実験を行います：

- **挙動検証**: サブエージェント、スキル、コマンドの動作確認
- **比較実験**: 複数のアプローチを比較
- **効果測定**: 機能の効果を定量的に評価
- **再現性確認**: 過去の観察を再検証

## 新しい実験を始める

1. templates/ から適切なテンプレートを選択
2. 新しいリポジトリを作成
3. 実験を実施
4. このREADMEの表を更新してPR
