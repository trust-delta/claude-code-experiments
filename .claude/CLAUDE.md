# Claude Code Experiments

このリポジトリはClaude Codeの各機能（サブエージェント、スキル、カスタムスラッシュコマンド）の挙動を実験・検証するためのものです。

## 現在の実験

実験01: 対話的タスクにおけるサブエージェントの挙動検証
- タスク: TODOアプリの「優先度自動調整機能」の要件定義
- 目的: サブエージェントが対話的タスクに適しているか検証

## 実験用ツール

### サブエージェント
- requirements-definer: 要件定義用のサブエージェント

### スキル
- spec-helper: 仕様書作成支援スキル

## 実験の進め方

各実験は experiments/ 配下にディレクトリを作成し、
3つのパターン（サブエージェント、メイン+スキル、メイン単体）で
同じタスクを実行して比較します。

詳細は experiments/01-subagent-dialog-tasks/README.md を参照してください。
